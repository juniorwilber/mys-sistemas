# MYS Sistemas 🖥️

Landing page para **MYS Sistemas**, una tienda de tecnología y servicio técnico ubicada en La Convención, Cusco. El sitio presenta sus servicios (reparación de impresoras, computadoras y laptops), la venta de equipos de marcas como HP, Lenovo, ASUS, Epson y Canon, y su propuesta de diagnóstico honesto con atención el mismo día.

**Qué resuelve:** le da al negocio una presencia web profesional que muestra sus servicios de soporte técnico y su catálogo de equipos, genera confianza en los clientes de la zona y facilita el contacto para diagnósticos y ventas.

---

## 🧰 Tecnologías

- **HTML5** — estructura y contenido semántico.
- **CSS3** — diseño responsivo, variables y animaciones.
- **JavaScript** — interactividad del lado del cliente.

## 🚀 Cómo verlo localmente

Es un sitio estático: basta con abrir `index.html` en el navegador. Para que todo cargue correctamente, se recomienda un servidor local:

```bash
# Con Python instalado
python -m http.server 8000
# Luego abre http://localhost:8000
```
